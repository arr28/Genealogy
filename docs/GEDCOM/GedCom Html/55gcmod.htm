The GEDCOM Standard Release 5.5: Data Model Chart


The PostScript version of the data model chart was available by anonymous ftp at ftp.gedcom.org in the directory /pub/genealogy/gedcom. The files are no longer available there. The best source for the data model charts was the Adobe Acrobat version of the specification which is at:

http://writedoc.com/lds/gedcom/gedcom55.pdf

That file is also no longer available.

You will need the Adobe Acrobat Reader to view PDF files. It is available for free at:

http://www.adobe.com/products/acrobat/readstep.html
